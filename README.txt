Download and install the nodejs from https://nodejs.org/en/download 
You can download nodejs version: node-v18.17.0-x64.msi file.
once downloaded install it.

Download the zip folder of the project from github and once downloaded unzip it and open in code editor preferably vs code editor.
now run the 'npm install' command.

once it's done you can run the command: 'npm start'
This starts the project in local. 

Please note that if npm install giving error about dependencies then run the following command:
'npm install --legacy-peer-deps'
